# assistant-language-eval-history

This branch is **machine-managed**. It is the durable store for
the nightly assistant language drift eval. Do not edit it by
hand; the next scheduled run will overwrite `TREND.md` and
append rows to `history.csv`.

See `data/assistant-language-eval/TREND.md` for the trailing
30-day pass-rate dashboard. The `docs/assistant-language-eval.md`
file on the default branch documents how the eval works.
